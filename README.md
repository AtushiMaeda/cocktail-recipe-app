# カクテルレシピアプリ

PC・スマートフォンで見れるレスポンシブなカクテルレシピアプリです。

## 機能

- 📱 レスポンシブデザイン（PC・スマートフォン対応）
- 🍸 100種類以上の有名なカクテルレシピを収録
- 🔍 カクテル名、カテゴリー、アルコール度数で検索・フィルタリング
- 📝 オリジナルカクテルレシピの追加機能
- 💾 追加したレシピはローカルストレージに保存

## 技術スタック

- React + TypeScript
- Vite
- Tailwind CSS

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build
```

## 使い方

1. アプリケーションを起動すると、カクテル一覧が表示されます
2. 検索バーやフィルターを使用してカクテルを探せます
3. カクテルカードをクリックすると詳細情報が表示されます
4. 「レシピを追加」ボタンからオリジナルカクテルを追加できます

## 注意事項

- オリジナルカクテルのデータはブラウザのローカルストレージに保存されます
- ブラウザのデータを削除すると、追加したカクテルも削除されます
